# ShellXss
 
> Versão: 1.0
 
Esse script abre uma sessão shell em um local vulneravel a injeção xss

# Plataformas 
✔️ Kali Linux 💻

# Como instalar
 
+ apt-get install git -y
 
+ git clone https://github.com/H7XX/ShellXss
 
+ cd ShellXss/
 
+ chmod +x install.sh
 
+ ./install.sh
 
para executar: 
+ python3 shellxss.py
